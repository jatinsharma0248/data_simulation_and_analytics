# Business Data Simulation & Analysis

## Overview

This project demonstrates the ability to design structured datasets, 
simulate realistic business data, and perform analytical queries to generate insights. 
The focus is on clear data modeling, logical assumptions, and well-structured queries that support meaningful analysis.

![Dashboard](snapshots/Dashboard%20.png)

## Objectives

* Create a realistic business dataset
* Design a clean and logical data structure
* Write analytical queries to explore patterns
* Derive insights that support business decisions

## Dataset

The dataset simulates a business environment and may include entities such as customers, policy_tenure , vehicle_id.
Data is structured to reflect realistic relationships between these entities.

## Analysis

SQL queries (or other analytical tools) are used to:

* Analyze trends and patterns
* Answer business-related questions
* Generate insights from the dataset

## Key Skills Demonstrated

* Data modeling and dataset design
* Business data simulation
* Analytical querying
* Insight generation from structured data

# Tools Used 
* SQL for Data Generation
* Power BI for Dashboard
